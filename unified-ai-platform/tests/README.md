# Unified AI Platform - Test Suite

## Overview

Comprehensive test suite for the Unified AI Platform with 233+ tests covering unit tests, integration tests, security validation, and performance benchmarks.

## Quick Start

```bash
# Install dependencies
cd unified-ai-platform
npm install

# Run all tests
npm test

# Run with coverage
npm test -- --coverage

# Run specific test file
npm test -- tests/unit/index.test.js

# Run in watch mode
npm test -- --watch
```
